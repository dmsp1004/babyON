# babyON Project - Architecture Documentation

## Project Overview

**babyON** is a childcare matching platform that connects parents seeking childcare services with babysitters. This is a **monorepo** containing both backend and frontend components.

**Project Name**: babyON - 아이 돌봄 매칭 플랫폼 (babyON - Child Care Matching Platform)

---

## Project Structure

This is a **monorepo** with the following structure:

```
babyON/
├── childcare/          # Spring Boot Backend API
├── babyon_app/         # Flutter Mobile/Web Frontend
└── lib/                # External libraries (Lombok JAR files)
```
